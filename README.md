# seos\_sandbox

Common base and layout for SeOS applications

## Getting Started

The project does not build, it is just a collection of submodules from wich the base layout gets prepared.

### Dependencies

* sel4
* muslc
* utils
* sel4muslcsys
* sel4platsupport
* sel4utils
* sel4debug
* sel4allocman
* camkes
* capdl
* seos\_libs
